MECHANISM 1

Intelligent Normalization

M1 is not a simple spell-checker—it's a specialized normalization algorithm designed specifically for neurodiverse text patterns.

Key Algorithm: Damerau-Levenshtein

Unlike standard Levenshtein distance, Damerau-Levenshtein specifically accounts for adjacent transpositions (e.g., 'teh' → 'the' as ONE edit, not two).

This is mathematically critical for addressing dyslexic writing—transpositions are the signature error pattern.

Safety Guardrails

  • ✓ Skip correction if token is already valid English
  • ✓ Preserve proper nouns and domain-specific terms
  • ✓ Edit distance threshold prevents over-correction
def normalize_text(text):
  # Tokenize input
  tokens = text.split()
  corrected = []

  for word in tokens:
    # Skip if already valid
    if word.lower() in english_words:
      corrected.append(word)
      continue

    # Find best match (Damerau-Levenshtein ≤ 2)
    best = find_closest(word, threshold=2)
    corrected.append(best or word)

  return " ".join(corrected)

MECHANISM 2

Semantic Evaluation

The core of our solution is a fine-tuned DistilBERT model that looks beyond keywords to understand context and meaning.

Why Transformers?

Traditional graders use "Bag of Words" approaches which fail when words are misspelled. Transformers use attention mechanisms to understand the relationship between words, even if some are noisy.
