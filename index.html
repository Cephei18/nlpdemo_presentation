1. Problem Statement

The educational assessment landscape currently penalizes neurodiverse students for surface-level errors rather than evaluating their actual understanding.

Traditional automated graders rely on rigid keyword matching, creating a double penalty for dyslexic students:

  • Spelling Penalty Misspelled keywords flagged as incorrect despite correct concepts.
  • Semantic Failure Standard BERT models fail to tokenize "noisy" text, losing context entirely.

3. Methodology (M1 ‚Üí M2)

M0: Acquisition

Robust entry point handling raw student submissions. We implemented a dual-mode system:

  • Google Vision API Ready
  • Direct Typed Input
  • Stable Simulation Mode

M1: Normalizer

We utilize Damerau-Levenshtein Distance instead of standard edit distance. This algorithm mathematically accounts for adjacent transpositions (e.g., 'teh' ‚Üí 'the'), the signature error of dyslexia.
