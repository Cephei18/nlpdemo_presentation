# Dyslexia-Aware Semantic Grader

A demonstration of a semantic grading system designed to be aware of dyslexia patterns.

## Live Demo
[View the Live Demo](https://cephei18.github.io/nlpdemo/)

## Features
- **Dyslexia Simulation**: Visualizes how text might appear to someone with dyslexia.
- **Semantic Analysis**: Uses NLP to grade content based on meaning rather than just surface-level mechanics.
- **Ablation Study**: Visual comparison of model performance.

## Usage
Open `index.html` in your browser or visit the live demo link above.
